# jquery 简介

中文简单教学(支持文档查询)：https://www.jquery123.com/

## 什么是jquery
jquery是一个js的工具库

## jquery的作用
jquery有以下主要功能：

针对页面元素，jquery提供了很多操纵页面元素的函数

还有一些工具函数，方便一些运算

jquery可以充当发起网络请求的客户端工具

## 安装
在官网上点击download进行下载

cdn: 共享资源节点

## 在页面中的使用
```html
<!DOCTYPE html>
<html>
    <head></head>
    <body></body>
    <!-- 通过脚本形式进行引入 -->
    <script src="./jquery.min.js"></script>
    <script>
        // 在引入jquery后，此处就可以使用jquery的功能了
    </script>
</html>
```
